7. POSITION-TO-TEMPLATE MAPPING RULES
================================================================================

When the upstream team maps raw brokerage data to CSV:

  Produto = "CDB" + Estratégia = "Pós-fixado"  → cdb_cdi_generic
  Produto = "CDB" + Estratégia = "Inflação"     → cdb_ipca_generic
  Produto = "CDB" + Estratégia = "Pré-fixado"   → cdb_pre_generic
  Produto = "CRA" + Estratégia = "Pós-fixado"   → cra_cdi_generic
  Produto = "CRA" + Estratégia = "Inflação"     → cra_ipca_generic
  (same pattern for CRI, CDCA, LCI, LCA)

  Produto = "Debênture" + IR=0                   → debenture_*_incentivada
  Produto = "Debênture" + IR>0                    → debenture_*_regular

  TESOURO DIRETO - LFT                           → tesouro_selic_generic
  TESOURO DIRETO - NTN-B-P                        → tesouro_ipca_principal_generic
  TESOURO DIRETO - NTN-B                          → tesouro_ipca_juros_generic
  TESOURO DIRETO - LTN                            → tesouro_prefixado_generic
  NTN-B / NTNB (secondary)                        → tesouro_ipca_juros_generic
  NTN-F (secondary)                               → tesouro_ntnf_generic

  Listed equity ticker (e.g., VALE3)              → equity_vale3
  BDR ticker (e.g., TSLA34)                       → bdr_tsla34
  ETF ticker (e.g., BOVA11)                       → etf_bova11
  FII ticker (e.g., HGLG11)                       → fii_hglg11
  Fund name                                       → fund_<slug>
  Crypto token                                    → crypto_<name>
  COE (any)                                       → coe_generic

  Unseen equity ticker                            → equity_br_listed_generic
  Unseen BDR ticker                               → bdr_equity_listed_generic
